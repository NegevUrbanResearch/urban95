Cities for Children

Beer Sheva Accessibility Map

  • Building Colors

    Each building is colored based on how many amenities are nearby. Green indicates high accessibility with many amenities within walking distance, while red indicates fewer nearby options.

  • Analyze a Building

    Click anywhere on the map to select the nearest building. A circle will appear showing the analysis radius, and you'll see a count of all amenities within that area in the popup panel below.

  • Filter Amenity Types

    Use the filter dropdown to focus on specific types of amenities such as healthcare, education, or recreation.

  • Adjust Analysis Radius

    Use the radius slider to change the walking distance used for analysis, from 50 meters up to 500 meters.

This map is a collaboration between Urban95 and NUR as part of the "Cities for Children" initiative.

Methodology

Each building's accessibility score is calculated by counting amenities within the specified walking radius. Trees are included in the analysis but weighted at 1/4 the value of other amenities.

Data

The map uses building footprints, points of interest including healthcare facilities, educational institutions, commercial areas, and recreational spaces, as well as street tree locations throughout the city.
